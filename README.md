# Medical Data Visualizer

This project analyzes medical examination data using Python, Pandas, Matplotlib, and Seaborn.

## Files
- medical_data_visualizer.py : Main analysis and visualization code
- medical_examination.csv : Dataset used for analysis

## Description
The project visualizes relationships between cardiovascular disease and medical indicators such as cholesterol, glucose, BMI, smoking, alcohol intake, and physical activity.

This project is part of the freeCodeCamp Data Analysis with Python certification.
